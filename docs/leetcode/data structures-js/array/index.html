array

前置相关知识​

（相关笔记的老师指路B站UP爱学习的饲养员） 算法的时间复杂度：算法的执行效率，指算法执行时间和输入值之间的关系。常用大O表示法， O(1)< O(logN)典型代表二分查找法< O(N)< O(NlogN)排序< O(N^2)< O(2^N)< O(N!)

算法的空间复杂度：指算法的数据存储空间和输入值之间的关系。 只有常量通常O(1)< 递归O(N)< < O(N^2)

时空二选一工作时优先时间，面试时和面试官讲清楚两种方法区别（如果有）

介绍​

数组：连续的内存空间中，存储一组相同类型的元素

区分​

访问：通过索引找到某元素的值， 搜索：直接找某元素，需要遍历，和索引无关，通过值找索引

基本方法的时间复杂度（分析数据结构优劣）​

1.访问O(1)

2.搜索O(N)

3.插入O(N)

4.删除O(N)

特点​

读多写少

常用操作​

1.创建数组​

常规方式:

var myCars=new Array();
myCars[0]="Saab";      
myCars[1]="Volvo";
myCars[2]="BMW";
  • 简洁方式:
var myCars=new Array("Saab","Volvo","BMW");
  • 字面量:
var myCars=["Saab","Volvo","BMW"];
  • 从字符串构建数组:
const fruits3 = 'Apple, Banana'.split(', ');
console.log(fruits3.length);

2.添加元素​

  • 末尾添加
const fruits = ['Apple', 'Banana'];
const newLength = fruits.push('Orange');
console.log(fruits);
// ["Apple", "Banana", "Orange"]
console.log(newLength);
// 3
  • 开头添加
const fruits = ['Banana', 'Mango'];
const newLength = fruits.unshift('Strawberry');
console.log(fruits);
// ["Strawberry", "Banana", "Mango"]
console.log(newLength);
// 3
  • splice() 方法：在指定位置删除或添加元素。
let arr = [1, 2, 3, 4, 5];
// 在数组末尾添加元素
arr.splice(arr.length, 0, 6, 8, 9); // 第二个参数为0的时候，是可以添加参数的，1的时候是删除

3.访问元素​

 const fruits = ['Apple', 'Banana'];

// 数组第一个元素的索引始终为 0。
fruits[0]; // Apple

// 数组第二个元素的索引始终为 1。
fruits[1]; // Banana

// 数组最后一个元素的索引总是比数组的长度小 1。
fruits[fruits.length - 1]; // Banana

// 使用大于数组长度的索引会返回"undefined"
fruits[99]; // undefined

4.修改元素​

  • 直接使用索引修改元素：使用数组的索引来访问和修改元素。array[0] = "新元素";
  • splice() 方法：splice() 方法可以用于添加或删除数组中的元素。将数组中的第一个元素更改为“新元素”，array.splice(0, 1, "新元素"); 例子：
 const fruits = ['Apple', 'Banana', 'Strawberry'];
const start = -2;
const deleteCount = 2;
const removedItems = fruits.splice(start, deleteCount, 'Mango', 'Cherry');
console.log(fruits);
// ["Apple", "Mango", "Cherry"]
console.log(removedItems);
// ["Banana", "Strawberry"]

5.删除元素​

const fruits = ['Apple', 'Banana', 'Strawberry', 'Mango'];
const start = 1;
const deleteCount = 2;
const removedItems = fruits.splice(start, deleteCount);
console.log(fruits);
// ["Apple", "Mango"]
console.log(removedItems);
// ["Banana", "Strawberry"]
  • 删除第一个
const fruits = ['Apple', 'Banana'];
const removedItem = fruits.shift();
console.log(fruits);
// ["Banana"]
console.log(removedItem);
// Apple
  • 删除最后一个
const fruits = ['Apple', 'Banana', 'Orange'];
const removedItem = fruits.pop();
console.log(fruits);
// ["Apple", "Banana"]
console.log(removedItem);
// Orange

6.遍历数组​

  • map()：通过指定函数处理数组中的每一个元素，并返回一个处理后的新数组。
const array1 = [1, 4, 9, 16];

// Pass a function to map
const map1 = array1.map(x => x * 2);

console.log(map1);
// Expected output: Array [2, 8, 18, 32]
  • for 循环:
eg:
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}

eg:
const fruits = ['Apple', 'Mango', 'Cherry'];
for (const fruit of fruits) {
  console.log(fruit);
}
// Apple
// Mango
// Cherry
  • foreach(元素，索引，数组)
 const fruits = ['Apple', 'Mango', 'Cherry'];
fruits.forEach((item, index, array) => {
  console.log(item, index);
});
// Apple 0
// Mango 1
// Cherry 2

  • 使用 filter() 方法遍历数组并返回符合条件的新数组，例如：
const newArray = arr.filter(function(item) {
  return item > 2;
});
  • 使用 reduce() 方法遍历数组并返回一个值，例如：
const sum = arr.reduce(function(total, item) {
  return total + item;
}, 0);

7.查找元素​

  • 使用 indexOf() 方法查找数组中是否有特定值，例如：
const array = [1, 2, 3];
console.log(array.indexOf(2)); // 1
  • 使用 find() 方法查找第一个符合条件的数组元素，例如：
const array = [1, 2, 3];
const found = array.find(function(element) {
  return element > 1;
});
console.log(found); // 2
  • 使用 filter() 方法查找满足特定条件的数组中的所有项目，例如：
const array = [1, 2, 3];
const greaterThanOne = array.filter(function(element) {
  return element > 1;
});
console.log(greaterThanOne); // [2, 3]
  • 使用 includes() 方法检查数组中是否有特定值，例如：
const array = [1, 2, 3];
console.log(array.includes(2)); // true

8.数组的长度​

arr.length()

9.数组排序（内置的排序方法）​

  • 使用 sort() 方法对数组进行排序，sort默认按unicode排序，arr.sort(function(a,b){return a-b;升序})b-a降序，影响原数组.例如：
const array = [3, 2, 1];
array.sort();
console.log(array); // [1, 2, 3]
  • 使用 reverse() 方法反转数组中元素的顺序，例如：
const array = [1, 2, 3];
array.reverse();
console.log(array); // [3, 2, 1]

练习题​

485最大连续1的个数

var findMaxConsecutiveOnes = function(nums) {
    let maxCount = 0, count = 0;
    const n = nums.length;
    for (let i = 0; i < n; i++) {
        if (nums[i] === 1) {
            count++;
        } else {
            maxCount = Math.max(maxCount, count);
            count = 0;
        }
    }
    maxCount = Math.max(maxCount, count);
    return maxCount;
};


283移动0 所有0移动到数组末尾

var moveZeroes = function(nums) {
        //获取数组长度
        let len = nums.length;
        //循环遍历数组
        for(let i =0;i<len;i++){
            //当数组哪个位置是0 ，就删除这个这个位置的元素，然后在末尾补 0
            //从而就达到了 把所有0移动到数组末尾的要求且保持的非零元素的相对顺序
            if(nums[i]==0){
                nums.splice(i,1)
                nums.push(0)
            //当为0的元素删除后，下一个元素就会前进一位占据该位置，所以要从该位置在进行判断
                i--
            //当移动到末尾的元素，就不用再一次进行遍历了，所以遍历的长度要减去1位
                len--
            }
        }
        return nums
}

27移除元素

//双指针算法
function removeElement(nums: number[], val: number): number {
    let left = 0
    let right = nums.length
    while(left < right) {
        if (nums[left] === val) {
                    // 优化: 找到右边复合条件值的索引
            while(nums[right-1] === val) {
                right--
            }
            if (right<left) {
                break
            }
            nums[left] = nums[right-1]
            right--
        } else {
            left++
        }
    }
    return left
};
