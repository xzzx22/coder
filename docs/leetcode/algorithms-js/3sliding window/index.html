3sliding window

介绍​

滑动窗口：可以解决数组中的定长问题，如长度为3的和。

特点​

目的是减少while循环

基本方法的时间复杂度（分析算法优劣）​

O(N)

练习题​

209长度最小的子数组

 
var minSubArrayLen = function(target, nums) {
    // 长度计算一次
    
   const len = nums.length;
    if (len == 0 && nums == null) return 0;
    let l = r = total = 0,
        res = len + 1; // 子数组长度最大不会超过自身
        
    while(r < len) {
        total += nums[r];//长度和
        r+=1;
        // 窗口滑动
        while(total >= target) {
            // r始终为开区间 [l, r)
            res = Math.min(res,r-l);
            total-=nums[l];
             l+=1;
        }
    }
    return res > len ? 0 : res;
};
时间复杂度：O(n)
空间复杂度：O(1)
一些录友会疑惑为什么时间复杂度是O(n)。

不要以为for里放一个while就以为是O(n^2)啊， 主要是看每一个元素被操作的次数，每个元素在滑动窗后进来操作一次，出去操作一次，每个元素都是被操作两次，所以时间复杂度是 2 × n 也就是O(n)。
